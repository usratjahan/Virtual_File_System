# Virtual_File_System
This project simulates a simple Virtual File System (VFS) using C language.
Users can create, read, write, rename, delete, move, copy, search, and open/close files.
Each file has details like:
Name, folder, owner
Status (open or closed)
Includes a login system with user roles:
Regular users manage their own files
Admins access all files
Tracks system usage: total storage & number of open files.
